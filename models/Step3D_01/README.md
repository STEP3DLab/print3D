# Step3D_01

Единая папка образца для сайта Step3D.Lab.

Содержимое:
- `Step3D_01_reference.scad` - параметрическая 3D-модель образца для публикации и быстрой правки
- `Step3D_01_preview.svg` - визуальный эскиз образца для сайта
- `Step3D_01_chertezh.svg` - упрощенный чертеж с основными размерами

Назначение папки:
- убрать дублирование ссылок на внешний репозиторий
- держать материалы образца в одном месте внутри проекта сайта
- использовать эту папку как единую точку входа для студентов

Примечание:
Точный производственный бинарный STL-файл архива исследования в текущем сеансе не был перенесен в репозиторий через GitHub-коннектор. Для сайта и учебной навигации опубликована параметрическая референс-модель в формате SCAD и визуальные материалы образца.
